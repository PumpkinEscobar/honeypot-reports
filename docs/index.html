What this is

A SOC home lab built to study real attacker behavior. The sensor runs on AWS and records unsolicited traffic off the open internet: the background noise of scanners, botnets, and opportunistic intrusion attempts.

The pipeline

dionaea→wazuh→opensearch→python

Captured events flow from the honeypot into a Wazuh SIEM, get indexed in OpenSearch, then are analyzed and rendered to a static report with Python.

What the report covers

  • 01Hit volume, unique sources, high-frequency offenders, probable scanners
  • 02Protocol targeting breakdown
  • 03Hourly attack timeline
  • 04Top 10 attacker IPs with risk flags
  • 05MITRE ATT&CK technique mapping with live counts
  • 06Most recent raw events
